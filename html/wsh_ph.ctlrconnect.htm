ph.ctlrconnect Windows Script Host function
Description
Connects PowerHome to either an IR, X10, or Other controller.
Syntax
ph.ctlrconnect ( type, num )
Argument Description
type A string representing the type of the controller to connect to. Valid values are “IR”, “X10”, and "Other".
num An integer representing the number of the controller to connect to. Valid values are 1 thru 5 for each of the 5 IR, X-10, or Other controllers.
Return value
Integer. Returns a value dependent upon the particular controller defined. A 0 will indicate a successful connect. Any other value indicates an error.
