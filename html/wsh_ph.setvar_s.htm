ph.setvar_s Windows Script Host function
Description
Sets the value of a LOCAL, TEMP, or GLOBAL system variable with a string value.
Syntax
ph.setvar_s ( n, n, s )
Argument Description
n The type of system variable you would like to set. Valid values are 1 for LOCAL variables, 2 for TEMP variables, and 3 for GLOBAL variables.
n The number of the appropriate system variable. Valid values are 1 – 10 for LOCAL and TEMP variables and 1 – 20 for GLOBAL variables.
s A string you would like stored within the specified variable.
Return value
Integer. Returns a 0 if successful and a 1 the system variable is undefined.
