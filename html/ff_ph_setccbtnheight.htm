ph_setccbtnheight PowerHome formula function
Description
Changes the height of a Control Center button.
Syntax
ph_setccbtnheight ( id, seq, height )
Argument Description
id The ID of the Control Center tab containing the button.
seq The sequence value of the button.
height The height of the button.
Return value
Integer. Returns 0.
Usage
See ph_setccbtn for more details.
