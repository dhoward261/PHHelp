ph.getccobjgraphic Windows Script Host function
Description
Retrieves the current graphic file in use by a Control Center graphic object.
Syntax
ph.getccobjgraphic ( tabid, id, type )
Argument Description
tabid String. The tab ID containing the graphic object.
id String. The ID of the Control Center graphic object.
type Integer. The type of graphic file to be returned. Use 0 for the "up" graphic file, 1 for the "down" graphic file.
Return value
String. Returns the full path and filename of the specified graphic object. Returns "- 1" if the tabid / id cannot be found.
