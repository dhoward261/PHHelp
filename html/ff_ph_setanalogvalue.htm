ph_setanalogvalue PowerHome formula function
Description
Allows the setting of the raw value, the calc value, and the calc message along with the optional firing of any triggers of Analog device types.
Syntax
ph_setanalogvalue ( id, value, calcvalue, calcmsg, flags )
Argument Description
id String. The ID of the analog device type.
value Long. The raw value to set the analog device type to.
calcvalue String. The calc value to set for the specified analog device.
calcmsg String. The calc message to set for the analog device.
flags Integer. The flags for controlling which values are used and how the function performs.
Return value
Integer. Returns 0 if successful. Returns 1 if the ID cant be found and 2 if an error occurs while updating the analogio table.
Usage
Use this function to directly set the values of an analog device type without using a controller.

The flags parameter is the sum of individual flag values. These flag values are:

1 - Update the raw value with the supplied raw value.
2 - Update the calc value with the supplied calc value.
4 - Update the calc message with the supplied calc message.
8 - Update the Device Status screen and any Remote DS clients with the new values.
16 - Fire any triggers associated with the updating of the specified analog device type.

The proper flags value to update an analog device's raw value, calc value, and update the Device Status screen would be 11 (1 + 2 + 8). No triggers will be fired and the calc message will be unchanged. 32 - Apply the calctoraw formula to the calcvalue and set the raw value.
64 - Apply the rawtocalc formula to the value and set the calcvalue.
