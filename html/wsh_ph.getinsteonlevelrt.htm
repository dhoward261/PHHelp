ph.getinsteonlevelrt Windows Script Host function
Description
Retrieves the current status (level) of an Insteon device is realtime.
Syntax
ph.getinsteonlevelrt ( id )
Argument Description
id The ID of the Insteon device whose status/level you wish to retrieve.
Return value
Double. Returns the current level of the specified Insteon device. 0 means that the Insteon device is off. 1 to 255 means that the device is on and indicates the current Insteon level.
Usage
Use this function to query the Insteon device and retrieve the actual current status. This function may have a slight delay while the device is queried.
