ph.setglobal_t Windows Script Host function
Description
Sets the value of a global variable with a time value.
Syntax
ph.setglobal_t ( s, t )
Argument Description
s A string identifying the ID of the global variable to set.
t A time value which you would like assigned to the specified global variable.
Return value
Integer. Returns a 0 if successful and a 1 if an error occurred while updating the global variable. The value to be stored must be a time value. All global variables are stored internally as strings and will be converted. This value will have a mask of ‘hh:mm:ss.fff’ applied.
