ph.setccobjx Windows Script Host function
Description
Changes the x value of a Control Center object.
Syntax
ph.setccobjx ( tabid, id, x )
Argument Description
tabid The ID of the Control Center tab containing the object.
id The ID of the object.
x The x value of the object. This is the horizontal position of the upper left corner of the object.
Return value
Integer. Returns 0.
Usage
See ph.setccobj for more details.
