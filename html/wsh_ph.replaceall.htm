ph.replaceall Windows Script Host function
Description
Replaces all occurrences of a substring within a string with another substring.
Syntax
ph.replaceall ( s, s, s )
Argument Description
s The string in which you want to search and replace substrings.
s The substring you want to search for.
s The substring you want to replace with.
Return value
String. Returns the string with the replacements made.
