freedb
This section is for setting up freedb.  freedb is an online database of CD's and allows you to place CD's into your computers CD player, generate a unique serial number, look that number up online in the freedb database, and save the disc and track information into the Discs section of your PowerHome database.  You can select the freedb server you wish to connect to but the default server freedb.freedb.com should be fine.  You will need to enter your email address for proper freedb queries to be built.  You should also check the Use Proxy Server box if you must use a proxy server to access the internet.  The proxy server values are defined in the Proxy section.
