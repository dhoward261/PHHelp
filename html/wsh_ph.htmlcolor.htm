ph.htmlcolor Windows Script Host function
Description
Returns the HTML hexadecimal equivalent of a combined RGB color.
Syntax
ph.htmlcolor ( n )
Argument Description
n A long value representing a color. This color value is determined by combining Red, Green, and Blue values ranging from 0 to 255 using the following formula: Red + (Green * 256) + (Blue * 65536).
Return value
String. Returns the HTML hexadecimal equivalent of the combined RGB color in the form of #RRGGBB. For example: If you had a color value of 255 (pure red) then the output from this function would be #FF0000.
