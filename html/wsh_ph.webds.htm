ph.webds Windows Script Host function
Description
Returns a dynamic HTML table containing the PowerHome Device Status screen.
Syntax
ph.webds ( type, nexturl )
Argument Description
type Integer. The type of web based Device Status screen to return. Use 0 for the new version with Control buttons on the left. Use 1 for the legacy version with Control buttons on the right. Use 2 for an alternative version with the Control buttons on the left.
nexturl String. A string representing the URL of the next page to open after processing a device status control button.
Return value
String. Returns a complete HTML table source code for the PowerHome Device Status screen at that moment. Use this function to quickly incorporate the Device Status screen directly into your own PSP code creations with virtually no coding. Using this function, you can create your own menus and mix or match individual PowerHome web elements.
