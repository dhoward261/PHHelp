ph_concatvar PowerHome formula function
Description
Concatenates a string to a system variable.
Syntax
ph_concatvar ( type, id, string )
Argument Description
type The type of the system variable. 1 denotes a local variable, 2 denotes a temp variable, and 3 denotes a global variable.
id The number of the system variable. 1 thru 10 are valid for local and temp variables. 1 thru 20 are valid for global variables.
string The string that you would like to concatenate to the system variable.
Return value
String. Returns the value of the system variable after the string has been concatenated to it.
