ph_setccbtny PowerHome formula function
Description
Changes the y value of a Control Center button.
Syntax
ph_setccbtny ( id, seq, y )
Argument Description
id The ID of the Control Center tab containing the button.
seq The sequence value of the button.
y The y value of the button. This is the vertical position of the upper left corner of the button.
Return value
Integer. Returns 0.
Usage
See ph_setccbtn for more details.
