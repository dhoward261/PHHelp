ph_convertir PowerHome formula function
Description
Converts raw IR code strings to other IR formats.
Syntax
ph_convertir ( convertto, code )
Argument Description
convertto String.  The type of IR code to convert to.
code String.  The IR code to convert.
Return value
String.  Returns the converted IR code in the specified format or an error message.
Usage
Valid convertto formats are:

  • "pronto"
  • "cir"
  • "howtek"
  • "rr2"
  • "slinke"
  • "uirtraw"
  • "uirtstruct"
  • "rr3"
  • "gc"

    You do not need to specify the input code format as the system will autodetermine it based upon the code itself.  If the function fails, it will return a string detailing the error.  All error strings start with "***ERROR:"
    • Examples
