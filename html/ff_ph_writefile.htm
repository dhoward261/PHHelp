ph_writefile PowerHome formula function
Description
Writes data to the specified file.
Syntax
ph_writefile ( s, n, s )
Argument Description
s A string representing the path and filename of the file you wish to write to.
n An integer representing how you wish to write to the file. A value of 0 will append the data to the file, any other value will cause the file to be overwritten.
s A string representing the data you wish to write to the file.
Return value
Integer. Returns a 0 if successful. Returns a 1 if an error occurred while opening the file.
