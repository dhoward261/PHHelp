ph.addtovar Windows Script Host function
Description
Adds a value to a system variable
Syntax
ph.addtovar ( type, id, value )
Argument Description
type The type of the system variable. 1 denotes a local variable, 2 denotes a temp variable, and 3 denotes a global variable.
id The number of the system variable. 1 thru 10 are valid for local and temp variables. 1 thru 20 are valid for global variables.
value A positive or negative double value that you want the system variable adjusted by.
Return value
Double. Returns the value of the system variable after it has been adjusted by value. This function should only be used when it is known that the system variable contains numeric data.
