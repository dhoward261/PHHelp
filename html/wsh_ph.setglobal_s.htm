ph.setglobal_s Windows Script Host function
Description
Assigns a string to the specified global variable.
Syntax
ph.setglobal_s ( s, s )
Argument Description
s A string identifying the ID of the global variable to set.
s A string which you would like assigned to the global variable.
Return value
Integer. Returns a 0 if successful and a 1 if an error occurred while updating the global variable.
