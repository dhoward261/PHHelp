ph_sendudp PowerHome formula function
Description
Uses a socket to send data using UDP.
Syntax
ph_sendudp ( url, port, data )
Argument Description
url The IP or URL to send UDP data to.
port The port on which to send UDP data.
data A string containing the UDP data to be broadcast.
Return value
Integer. Returns a 0.
Usage
Use this function to broadcast UDP data on the network. You may use the PowerHome UDP Listener plugin to listen for UDP traffic.
