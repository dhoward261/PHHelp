ph.getwindowstate Windows Script Host function
Description
Returns the state of the PowerHome frame window.
Syntax
Return value
Integer. Returns 1 if the PowerHome frame is minimized. Returns 2 if the PowerHome frame is maximized. Returns 0 if the PowerHome frame is in a normal state.
