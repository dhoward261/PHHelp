X10 Settings
Previous X10 Incoming Timeout (sec)
This parameter is used primarily for multi-x operations and controls what is the maximum time between incoming X10 signals before it is not considered part of a multi-x operation.
Clear X10 Status on Startup
Sets the X10 status variables to ‘Unknown’ whenever PowerHome is started.  If cleared, PowerHome will use the last known status.
Set X10 Status to Unknown on Status Request
Enabling this parameter will cause the internal X10 status variables and Device Status screen to display ‘Unknown’ for status whenever a status request command is issued to an X10 device that supports status request.
