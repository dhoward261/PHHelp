ph_getlasterror PowerHome formula function
Description
Returns the error number of the last error for certain functions.
Syntax
ph_getlasterror ( )
Return value
Integer. Returns the error number associated with the last error. Available after calling the following functions: ph_evalscript, ph_executescript, ph_runscript_0, ph_runscript_1, ph_runscript_2, ph_runscript_3, ph_runscript_4, ph_runscript_5
