ph.getsqlrows Windows Script Host function
Description
Returns the current number of rows in the specified data retrieval area.
Syntax
ph.getsqlrows ( n )
Argument Description
n The data retrieval area you wish to get the number of rows (1 – 10).
Return value
Long. Returns the number of rows currently within the data retrieval area. Same as the system variables SQLROWS1 thru SQLROWS10.
