ph.runscript_2 Windows Script Host function
Description
Executes a WSH (Windows Script Host) file.
Syntax
ph.runscript_2 ( s, s, n, n, s, a, a )
Argument Description
s The full path and filename of the WSH file you wish to execute.
s The language of the WSH file. Examples would be “vbscript” and “jscript”.
n Allow UI. Place a 1 here if you wish to allow User Interface elements to be created within the script. Place a 0 here to disallow user interface elements.
n Timeout. Specifies the time in milliseconds after which the user will be presented with an option to continue or discontinue execution of the script. A value of –1 means the script will never timeout. A value of 0 will fire the timeout event when it is detected that the script is hung.
s The function or subroutine which should be executed.
a The first parameter to be passed to the called function. Can be any datatype.
a The second parameter to be passed to the called function. Can be any datatype.
Return value
Integer. Returns 0 if successful. –1 is returned if the script file does not exist. –2 is returned if the script file cannot be opened. –3 is returned if an error occurred while initializing the Script Server. –4 is returned if a syntax error exists. –5 is returned if a runtime error is encountered.
