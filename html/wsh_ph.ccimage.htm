ph.ccimage Windows Script Host function
Description
Saves a Control Center tab to the clipboard or a graphic file.
Syntax
ph.ccimage ( id, type, copykeys, filename )
Argument Description
id String. The ID of the tab to convert to a graphic.
type Integer. The type of graphic to save as. Use 0 for the Clipboard, 1 for BMP, 2 for GIF, 3 for JPEG, and 4 for PNG. You may also use the enumerated values clipboard, bmp, gif, jpeg, png.
copykeys Integer. Use a value of 1 for the keys to be saved in the graphic. Use a value of 0 to not have the keys saved in the graphic.
filename String. The path and filename to save to. Use empty string for the clipboard.
Return value
Integer. Returns a 0.
Usage
Use this function to save a snapshot image of a Control Center tab to either the clipboard or a graphic file. Uses a bitblt function to accomplish the graphic creation.
