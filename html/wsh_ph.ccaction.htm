ph.ccaction Windows Script Host function
Description
Executes an action associated with a Control Center action object.
Syntax
ph.ccaction ( tabid, id, x, y )
Argument Description
tabid String. The tab ID containing the CC action object.
id String. The ID of the CC action object.
x Integer. The X coordinate where a click on an action object actually occurred. Use 0 if X is not important.
y Integer. The Y coordinate where a click on an action object actually occurred. Use 0 if Y is not important.
Return value
Integer. Returns 0 if successful. Returns 1 if the tabid and id combination cannot be found.
