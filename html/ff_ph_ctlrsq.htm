ph_ctlrsq PowerHome formula function
Description
Adds an item to a PowerHome controllers internal send queue using the internal communication routines.
Syntax
ph_ctlrsq ( id, command, parm1, parm2, parm3 )
Argument Description
id String. The ID of the controller to add queued command to.
command Integer. The type of command to queue. Common queued commands are 10 for IR play, 20 for X10 send, 30 for send Insteon direct, 31 for send Insteon group, 32 for send Insteon group cleanup, 51 for Set digital, 61 for Set analog out.
parm1 String. Parameter 1 as a string.
parm2 Integer. Parameter 2 as an integer.
parm3 Long. Parameter 3 as an integer.
Return value
Returns 0 if the function succeeds. Returns - 1 if the controller does not exist.
Usage
This function exposes the very base level of PowerHome controller communications and is provided to achieve the maximum level of control. Most actions can be achieved using the higher level PowerHome functions such as ph_x10, ph_ir, and ph_insteon.

An example of how this function may be used to send an X10 "On" command to housecode C would be: ph_ctlrsq("X10CTLRID",20,"C",102,0)
