ph.getx10level Windows Script Host function
Description
Retrieves the current level (dim / bright) value of the specifed X-10 ID.
Syntax
ph.getx10level ( s )
Argument Description
s A string representing the ID of the X-10 Device from which you wish to retrieve the level.
Return value
Double. Returns the level associated with the specified X-10 Device ID that PowerHome currently has stored. This is the equivalent of the system variable X10LEVEL## where ## is the house/unit code of the X-10 device.
