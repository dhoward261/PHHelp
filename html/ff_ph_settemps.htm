ph_settemps PowerHome formula function
Description
Allows for the quick initialization of temp variables [TEMP1] thru [TEMP5]
Syntax
ph_settemps ( temp1, temp2, temp3, temp4, temp5 )
Argument Description
temp1 Any. The value to assign to [TEMP1]. Data can be of any standard type.
temp2  Any. The value to assign to [TEMP2]. Data can be of any standard type.
temp3  Any. The value to assign to [TEMP3]. Data can be of any standard type.
temp4  Any. The value to assign to [TEMP4]. Data can be of any standard type.
temp5  Any. The value to assign to [TEMP5]. Data can be of any standard type.
Return value
Integer. Returns 0.
Usage
Use this function to rapidly set the first 5 TEMP variables. If you only want to set some of the TEMP variables, supply a string containing a single space for the variables you wish to leave unchanged.
Examples
The following examples demonstrate typical syntax/usage for this function.
ph_settemps(35,"Test"," ",2014-06-19,321.25)
The above example will set TEMP1, TEMP2, TEMP4, and TEMP5 to the specified number, string, date, and decimal respectively. TEMP3 will not be changed since it is set to a single space.
