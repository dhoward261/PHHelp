ph.regactivex Windows Script Host function
Description
Registers the activex controls (as the current PowerHome user) that are listed in the activex table.
Syntax
ph.regactivex ( )
Return value
Integer. Returns a 0.
Usage
Use this function to register the activex controls listed in the PowerHome activex table. This table default contains all the activex controls that are default used by PowerHome. This function is useful to allow PowerHome to run properly as a service. This function performs the same action as the menu option File|Register ActiveX.
