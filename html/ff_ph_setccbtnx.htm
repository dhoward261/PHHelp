ph_setccbtnx PowerHome formula function
Description
Changes the x value of a Control Center button.
Syntax
ph_setccbtnx ( id, seq, x )
Argument Description
id The ID of the Control Center tab containing the button.
seq The sequence value of the button.
x The x value of the button. This is the horizontal position of the upper left corner of the button.
Return value
Integer. Returns 0.
Usage
See ph_setccbtn for more details.
