ph_setccbtn PowerHome formula function
Description
Changes the attributes of a Control Center button. You can change all the attributes at once or individual ones depending upon the values you pass.
Syntax
ph_setccbtn ( id, seq, text, x, y, width, height, fcolor, bcolor, border )
Argument Description
id The ID of the Control Center tab containing the button.
seq The sequence value of the button.
text The text to place on the button. Use "" (empty string) to remove the text. Use a single space " " to not change the text.
x The x position of the button. This is the horizontal position of the upper left corner of the button. Use - 1 to not change the x position.
y The y position of the button. This is the vertical position of the upper left corner of the button. Use - 1 to not change the y position.
width The width of the button. Use - 1 to not change the width.
height The height of the button. Use - 1 to not change the height.
fcolor The foreground color of the button. Use -1 to not change the foreground color.
bcolor The background color of the button. Use -1 to not change the background color.
border The border of the button. 0 = No border, 2 = Box, 5 = lowered, 6 = raised, and 10 = button. Use - 1 to not change the border.
Return value
Integer. Returns 0.
Usage
Use this function or the individual ph_setcc functions to change the appearance of Control Center buttons on the fly. Using this technique, you can make Control Center buttons reflect the state of items. Any changes made to the Control Center in this manner are NOT permanent. These changes will be seen in the desktop Control Center as well as the web version of the Control Center.
