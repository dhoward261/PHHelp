ph.yp2u Windows Script Host function
Description
Returns the equivalent PowerHome units for the specified Y pixels.
Syntax
ph.yp2u ( x )
Argument Description
x A value representing the number of pixels (in the Y axis) you wish converted to units.
Return value
Integer. Returns the Y pixels converted to PowerHome units.
Usage
Within the PowerHome program, all x, y, height, and width values are stored in units. This function will convert pixels in the vertical plane (Y axis) to their equivalent PowerHome Y unit value.
