ph_getmd5hash PowerHome formula function
Description
Returns the standard MD5 hash for the specified string of data.
Syntax
ph_getmd5hash ( data )
Argument Description
data String. The data you'd like the MD5 calculated for.
Return value
String. Returns the MD5 hash as a string of hex digits.
Usage
Use this function to calculate MD5 hashes.
Examples
The following examples demonstrate typical syntax/usage for this function.
ph_getmd5hash("Test")
This example will return the following string: 0CBC6611F5540BD0809A388DC95A615B
