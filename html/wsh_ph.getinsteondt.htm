ph.getinsteondt Windows Script Host function
Description
Retrieves the datetime of the last status change for the specifed Insteon Device ID.
Syntax
ph.getinsteondt ( s )
Argument Description
s A string containing the ID of the Insteon Device from which you wish to retrieve the datetime of the last status change.
Return value
Datetime. Returns the datetime value of the last status change associated with the specified Insteon Device ID that PowerHome currently has stored.
