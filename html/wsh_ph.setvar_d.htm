ph.setvar_d Windows Script Host function
Description
Sets the value of a LOCAL, TEMP, or GLOBAL system variable with a date value.
Syntax
ph.setvar_d ( n, n, d )
Argument Description
n The type of system variable you would like to set. Valid values are 1 for LOCAL variables, 2 for TEMP variables, and 3 for GLOBAL variables.
n The number of the appropriate system variable. Valid values are 1 – 10 for LOCAL and TEMP variables and 1 – 20 for GLOBAL variables.
d The date value you would like to have stored.
Return value
Integer. Returns a 0 if successful and a 1 the system variable is undefined. You must pass the value to be stored as a date. All LOCAL, TEMP, and GLOBAL variables are stored internally as strings and this value will be converted. A mask of ‘yyyy-mm-dd’ will be applied.
