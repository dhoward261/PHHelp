ph_getinsteonstat PowerHome formula function
Description
Retrieves the current status of the specifed Insteon Device ID.
Syntax
ph_getinsteonstat ( id )
Argument Description
id  String. The Insteon Device ID from which you wish to retrieve the status.
Return value
Integer. Returns the status associated with the specified Insteon Device ID that PowerHome currently has stored. Valid values are 0 for Unknown, 1 for Off, and 2 for On. If the ID does not exist, then -1 is returned.
