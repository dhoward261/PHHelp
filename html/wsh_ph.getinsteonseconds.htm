ph.getinsteonseconds Windows Script Host function
Description
Retrieves the number of seconds elapsed since the last status change for the specifed Insteon Device ID.
Syntax
ph.getinsteonseconds ( s )
Argument Description
s A string representing the Insteon Device ID from which you wish to retrieve the seconds elapsed since last status change.
Return value
Long. Returns the number of elapsed seconds since the last status change associated with the specified Insteon Device ID that PowerHome currently has stored.
