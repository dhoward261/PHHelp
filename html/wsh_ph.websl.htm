ph.websl Windows Script Host function
Description
Returns a dynamic HTML table containing the standard PowerHome web status line.
Syntax
ph.websl ( )
Return value
String. Returns a complete HTML table source code for the PowerHome web interface status line. Use this function to quickly incorporate the PowerHome web status line directly into your own PSP code creations with virtually no coding. Using this function, you can mix or match individual PowerHome web elements to create your personalized web screens.
