ph.comsendchar Windows Script Host function
Description
Sends a byte out of the specified COM control.
Syntax
ph.comsendchar ( comnum, bytevalue )
Argument Description
comnum The number of the COM control you are referencing. Valid values are 1 thru 5.
bytevalue An integer representing the ASCII value of the byte to be sent. Valid values are 0 thru 255.
Return value
Integer. Returns 0 if the character is successfully sent. Returns 1 if a comnum < 1 or > 5 is specified. Returns 2 if the specified COM control is not valid. Returns 3 if the bytevalue is out of range.
