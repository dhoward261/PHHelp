ph_insteonprodlistimport PowerHome formula function
Description
This function imports Insteon product list data into the PowerHome database.
Syntax
ph_insteonprodlistimport ( filename )
Argument Description
filename String. The full path and filename of the file to import.
Return value
Long. Returns the number of rows imported or a negative number indicating an error occured. - 1 thru - 15 is an internal import error. - 9999 indicates the import was successful but the database updated failed. 
Usage
This function is used to bring Insteon product list data as published by SmartHome into the PowerHome database. It is an internal function and should be used as database corruption can occur.
Examples
The following examples demonstrate typical syntax/usage for this function.
ph_insteonprodlistimport("c:\powerhome\ipl20140605")
