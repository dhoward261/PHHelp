ph_processemail PowerHome formula function
Description
Processes and executes any Control Email using the default MAPI client .
Syntax
ph_processemail ( )
Argument Description
Return value
Integer. Returns a 0 if successful. Returns a 1 if an error occurs while logging into the mail system. Returns a 2 if an error occurs while retrieving email. Returns a 3 if an error occurs while reading the envelope. Returns a 4 if an error occurs while reading the message. Returns a 5 if an error occurs while deleting the email.
