ph.wcshow Windows Script Host function
Description
Causes the Web Center to either hide or display the open URL edit control.
Syntax
ph.wcshow ( type )
Argument Description
type A value of 0 will cause the URL edit control to be hidden. A value of 1 makes the URL edit visible.
Return value
Integer. Returns a 0 if successful. Returns a 2 if the Web Center is not open.
