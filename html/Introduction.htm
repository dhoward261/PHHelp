Introduction
PowerHome is an extremely powerful and flexible Windows based software package that will give you full control over your home’s lighting, appliances, infrared devices, and security when used in conjunction with the appropriate control hardware.
With PowerHome’s programmable interface, this control can be achieved via keyboard, mouse, web, Email, Insteon, UPB, X10, IR, Voice recognition, Socket communications, Windows Messaging, and even your internet enabled cell phone.
This user's manual is for version 2.1.5 of PowerHome. If this is your first time using PowerHome, we highly recommend that you first review the PowerHome2 QuickStart Guide.
Upgraders be sure and check out Appendix D for upgrade information and changes.
