ph_setccobjtext PowerHome formula function
Description
Changes the text of a Control Center object.
Syntax
ph_setccobjtext ( tabid, id, text )
Argument Description
tabid The ID of the Control Center tab containing the object.
id The ID of the object.
text The text of the object.
Return value
Integer. Returns 0.
Usage
See ph_setccobj for more details.
