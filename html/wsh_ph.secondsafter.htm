ph.secondsafter Windows Script Host function
Description
Returns the number of seconds one time occurs after another.
Syntax
ph.secondsafter ( time1, time2 )
Argument Description
time1 A time value that is the start time of the interval being measured
time2 A time value that is the end time of the interval
Return value
Double. Returns the number of seconds time2 occurs after time1. If time2 occurs before time1, a negative value is returned. Partial seconds are represented in decimal format.
