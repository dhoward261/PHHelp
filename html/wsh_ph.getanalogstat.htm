ph.getanalogstat Windows Script Host function
Description
Retrieves the current raw value of a Analog device in the Device Status Screen. This function DOES NOT cause the controller to retrieve the values and only returns what is currently stored within PowerHome.
Syntax
ph.getanalogstat ( id )
Argument Description
id The ID of the analog device to retrieve the raw value for.
Return value
Long. Returns the currently stored raw value for the specified analog device.
Usage
This function does not cause the controller to query the value. Use ph.getotheranalog to have the controller query the current actual value.
