ph.ctlrdisconnect Windows Script Host function
Description
Disconnects PowerHome from either an X-10, IR, or Other controller.
Syntax
ph.ctlrdisconnect ( type, num )
Argument Description
type A string representing the type of the controller to disconnnect from. Valid values are “IR”, “X10”, or "Other".
num An integer representing the number of the controller to disconnect from. Valid values are 1 thru 5 for each of the 5 IR, X-10 or Other controllers.
Return value
Integer. Returns a value dependent upon the particular controller defined. A 0 will indicate a successful disconnect. Any other value indicates an error.
