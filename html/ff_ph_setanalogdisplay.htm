ph_setanalogdisplay PowerHome formula function
Description
Sets the display values associated with records in the analogio table (Analog I/O screen) that will display in the Device Status screen.
Syntax
ph_setanalogdisplay ( id, foreground, background, graphic, refresh )
Argument Description
id The ID of the record in the analogio table.
foreground Long. The foreground color to display in the Control area of the Device Status screen. Use a negative number to use the existing value.
background Long. The background color to display in the Control area of the Device Status screen. Use a negative number to use the existing value.
graphic String. The path and filename of a graphic to display in the Control area of the Device Status screen. Use empty string "" to use the existing value.
refresh Integer. Use 0 to update the table but refresh the Device Status screen if open. Any other value will update the table as well as refresh the open Device Status screen.
Return value
Integer. Returns 0 if successful. Returns 1 if the ID does not exist. Returns 2 if the analogio table cannot be updated.
Usage
Since the Analog I/O screen only contains a single setting for foreground, background, and graphic, this function allows the user flexibility in changing these static values based upon the current value of the particular Analog I/O point. This function may be called in the CalctoRaw and RawtoCalc formulas defined for the Analog I/O point.
