ph.getccobjbcolor Windows Script Host function
Description
Retrieves the background color for the specified Control Center object.
Syntax
ph.getccobjbcolor ( tabid, id )
Argument Description
tabid The ID of the Control Center tab containing the object.
id The ID of the object.
Return value
Long. Returns the color value of the specified objects background. This value is equivalent to the value returned from the rgb() function. Returns - 1 if the object is not found.
Usage
Use the functions ph.getcclastobj and ph.getcclasttab to retrieve the tab ID and object ID of the last action object pressed in the Control Center. It is not necessary for the Control Center to be open in order to retrieve or make changes to objects.
