ph_getccobjborder PowerHome formula function
Description
Retrieves the border value for the specified Control Center object.
Syntax
ph_getccobjborder ( tabid, id )
Argument Description
tabid The ID of the Control Center tab containing the object.
id The ID of the object.
Return value
Integer. Returns the border value of the specified object. 0 is returned for no border, 2 is returned for box border, 5 is returned for a lower border, 6 is returned for a raised border, and 10 is returned for a button border. Returns - 1 if the object is not found.
Usage
Use the functions ph_getcclastobj and ph_getcclasttab to retrieve the tab ID and object ID of the last action object pressed in the Control Center. It is not necessary for the Control Center to be open in order to retrieve or make changes to objects.
