ph.directsql Windows Script Host function
Description
Execute SQL directly against the PowerHome database.
Syntax
ph.directsql ( sql )
Argument Description
sql A string containing a valid SQL statement. Inserts, Updates, and Deletes are all valid. See ph.sqlselect to perform Selects against the database.
Return value
Integer. Returns a 0 if the SQL was successfully executed. Otherwise a value equivalent to the SQL error when attempting to execute the SQL is returned.
