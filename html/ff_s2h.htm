s2h PowerHome formula function
Description
Hex encode a string or a file.
Syntax
s2h ( input, type )
Argument Description
input String. The string to hex encode or the full path and filename of a file to hex encode.
type Integer. The type of encoding to perform. Use 1 to hex encode the string in the input parameter. THis will be in unicode format. Use 2 to hex encode a file identified by the input parameter. Use any other value to hex encode the string in input in ANSI format.
Return value
String. Returns the hex encoded string. For every byte in a string or file, it will encode into 2 hex characters (effectively doubling the size of the data).
Usage
This function will allow you to convert binary data (such as from a file) and store it as a hexadecimal string for further processing within PowerHome.
Examples
The following examples demonstrate typical syntax/usage for this function.
s2h("c:\powerhome\phwiki.gif",2)
The above example will encode the file c:\powerhome\phwiki.gif into a hex string.

s2h("Test",0)
The above example will return "54657374"
