ph_prontotogc PowerHome formula function
Description
Converts a Pronto style Infrared code to a Global Cache compatible IR string.
Syntax
ph_prontotogc ( code, repeat )
Argument Description
code A string containing the Pronto IR code
repeat An integer representing the number of repeats to embed in the Global Cache IR String Use a negative number to search the Pronto code for a tag and use that value if present
Return value
String. The Global Cache combatible Infrared code.
