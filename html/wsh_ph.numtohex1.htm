ph.numtohex1 Windows Script Host function
Description
Converts an unsigned integer to a hexadecimal string and specify the returned length.
Syntax
ph.numtohex1 ( value, len )
Argument Description
value Unsigned Integer. The integer whose value you wish returned in hex. Valid values are from 0 to 65535.
len Integer. The number of hex digits that should be returned. Valid values are 1 thru 4.
Return value
String. Returns the number as a hexadecimal string of the specified length.
Usage
This function is essentially the same as the ph.numtohex() function coupled with the right() function.
