ph.setccobjy Windows Script Host function
Description
Changes the y value of a Control Center object.
Syntax
ph.setccobjy ( tabid, id, y )
Argument Description
tabid The ID of the Control Center tab containing the object.
id The ID of the object.
y The y value of the object. This is the vertical position of the upper left corner of the object.
Return value
Integer. Returns 0.
Usage
See ph.setccobj for more details.
