ph_ctlrcmd PowerHome formula function
Description
Sends a raw command to a PowerHome controller using the internal communication structure.
Syntax
ph_ctlrcmd ( ctlr, command, parm1, parm2, parm3, parm4 )
Argument Description
ctlr String. The ID of the controller to send the command to.
command Integer. The command to send to the controller.
parm1 Integer. Parameter 1 for the command.
parm2 Long. Parameter 2 for the command.
parm3 String. Parameter 3 for the command.
parm4 String. Parameter 4 for the command.
Return value
String. Returns the 4 return parameters from the communication structure. These values are separated with ~255 escape characters.
Usage
Use this command to send raw commands to PowerHome controllers. Details on the use of this function will be documented at a later date.
