ph_rtne PowerHome formula function
Description
Takes any numeric value and returns an empty string.
Syntax
ph_rtne ( n )
Argument Description
n Any numeric value
Return value
String. This function will always return an empty string. It is very handy within formulas within Send Keys commands and PSP pages when you are running a sequence of functions that return a numeric value. You can place all the functions added together within a ph_rtne function so that no extra keys or HTML is generated in the output.
