ph_windowstate PowerHome formula function
Description
Sets the state of the PowerHome frame window.
Syntax
ph_windowstate ( n )
Argument Description
n An integer representing the state the PowerHome frame should be set to. A value of 0 makes the frame normal, a value of 1 makes the frame minimized, and a value of 2 makes the frame maximized.
Return value
Integer. Returns a 0.
Usage
If in Kiosk mode (ph_setmode), ph_windowstate(0) will return a 1 and NOT set the window to normal. The only valid values for Kiosk mode are 1 and 2.
