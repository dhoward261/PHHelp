ph.chggrammar Windows Script Host function
Description
Changes the current grammar of the voice recognition engine.
Syntax
ph.chggrammar ( string )
Argument Description
string The grammar you would like to be the current grammar
Return value
Integer. Returns 0 if the grammar was changed successfully. Returns 1 if an error occurred while retrieving the grammar from the database. Returns 2 if no records exist in the grammar detail.
