ph_getupbstat PowerHome formula function
Description
Returns the status of the specified UPB device.
Syntax
ph_getupbstat ( id )
Argument Description
id String. The ID of the UPB device.
Return value
Integer. Returns the current status of the UPB device. 0 for Unknown, 1 for Off, 2 for On/Dimmed, 3 for Blinking.
