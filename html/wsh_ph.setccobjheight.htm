ph.setccobjheight Windows Script Host function
Description
Changes the height of a Control Center object.
Syntax
ph.setccobjheight ( tabid, id, height )
Argument Description
tabid The ID of the Control Center tab containing the object.
id The ID of the object.
height The height of the object.
Return value
Integer. Returns 0.
Usage
See ph.setccobj for more details.
