ph_enabletrigger PowerHome formula function
Description
Enables a disabled PowerHome trigger.
Syntax
ph_enabletrigger ( trigID )
Argument Description
trigID String. The ID of the trigger you wish to enable.
Return value
Integer. Returns a 0.
Examples
The following examples demonstrate typical syntax/usage for this function.
• ph_enabletrigger ( "LIVROOM_OFF" )
• ph_enabletrigger ( "[TEMP2]" ) --> where TEMP2 contains a DEVICE NAME
