ph_snapxml PowerHome formula function
Description
Extracts specified information from an XML string.
Syntax
ph_snapxml ( xml, value )
Argument Description
xml String. Contains the XML you'd like to parse.
value String. The XML tag whose value you'd like returned.
Return value
String. Returns the data between the opening and closing tags of the tag specified in the value parameter.
Usage
Use this function to extract tag information from XML data.
Examples
The following examples demonstrate typical syntax/usage for this function.
ph_snapxml(ph_readfile("c:\powerhome\test.xml"),"data1")
The above example will open and read the c:\powerhome\test.xml file and return the data between the <data1> and </data1> tags.
