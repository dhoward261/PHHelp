ph.setccobjborder Windows Script Host function
Description
Changes the border of a Control Center object.
Syntax
ph.setccobjborder ( tabid, id, border )
Argument Description
tabid The ID of the Control Center tab containing the object.
id The ID of the object.
border The border of the object. 0 = No border, 2 = Box, 5 = lowered, 6 = raised, and 10 = button.
Return value
Integer. Returns 0.
Usage
See ph.setccobj for more details.
