ph_insteonclearfailed PowerHome formula function
Description
Zeroes the "Failed Match" column of a specific Insteon device or all Insteon devices.
Syntax
ph_insteonclearfailed ( id )
Argument Description
id String. The ID of the Insteon device whose failed match column will be set to 0. Use empty string ("") to reset ALL Insteon devices.
Return value
Integer. Returns a 0.
