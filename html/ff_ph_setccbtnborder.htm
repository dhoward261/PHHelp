ph_setccbtnborder PowerHome formula function
Description
Changes the border of a Control Center button.
Syntax
ph_setccbtnborder ( id, seq, border )
Argument Description
id The ID of the Control Center tab containing the button.
seq The sequence value of the button.
border The border of the button. 0 = No border, 2 = Box, 5 = lowered, 6 = raised, and 10 = button.
Return value
Integer. Returns 0.
Usage
See ph_setccbtn for more details.
