ph.irchain Windows Script Host function
Description
Allows you to specify a chain of Infrared commands to be played sequentially for a specific piece of equipment.
Syntax
ph.irchain ( id, commands )
Argument Description
id The ID of the equipment whose IR chain you wish to play
commands A string containing the Key Descriptions of the IR commands you wish to play for the specified ID. Separate the key descriptions with a tab character (~t in PowerHome)
Return value
Integer. Returns a 0.
