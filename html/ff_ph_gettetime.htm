ph_gettetime PowerHome formula function
Description
Returns the DateTime for a Timed Event.
Syntax
ph_gettetime ( id, type )
Argument Description
id String. The ID of the Timed Event to return the DateTime for
type Integer. The type of DateTime to be returned. Use 0 for the Start Time. Any other value returns the Ref Time.
Return value
DateTime. The DateTime value for the specified Timed Event ID.
