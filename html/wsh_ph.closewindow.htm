ph.closewindow Windows Script Host function
Description
Sends both WM_CLOSE and WM_SYSCOMMAND windows messages to the supplied window handle to force the window closed.
Syntax
ph.closewindow ( handle )
Argument Description
handle Long. The handle of the window you wish to close.
Return value
Integer. Returns a 0.
Usage
Use the ph.findwindow function or it's derivatives to get the handle of the window you wish to close.
