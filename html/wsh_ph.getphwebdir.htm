ph.getphwebdir Windows Script Host function
Description
Returns the full path to the PowerHome webserver directory.
Syntax
ph.getphwebdir ( )
Return value
String. Returns the full path to the PowerHome webserver directory.
Usage
This function returns the full path to the PowerHome webserver root directory. This location is contained within the pwrhome.ini file and can be configured from the PowerHome Explorer under Setup->Web. The default is: c:\program files\powerhome\web
