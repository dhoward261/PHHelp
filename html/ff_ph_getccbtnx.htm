ph_getccbtnx PowerHome formula function
Description
Retrieves the x for the specified Control Center button. This is the horizontal position of the buttons upper left hand corner.
Syntax
ph_getccbtnx ( id, seq )
Argument Description
id The ID of the Control Center tab containing the button.
seq The sequence value of the button.
Return value
Long. Returns the x value of the specified button. Returns - 1 if the button is not found.
Usage
The sequence value is the sequence number assigned to a button in the Control Center Button Details screen in the PowerHome Explorer (in the explorer window, the sequence is actually multiplied by 10). Use the functions ph_getcclastbtn and ph_getcclasttab to retrieve the tab ID and sequence of the last button pressed in the Control Center. You can use the ph_lookupccbtn function to retrieve the first sequence of a button matching the passed parameters. It is not necessary for the Control Center to be open in order to retrieve or make changes to buttons.
