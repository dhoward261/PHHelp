ph.setccobjfcolor Windows Script Host function
Description
Changes the border of a Control Center object.
Syntax
ph.setccobjfcolor ( tabid, id, fcolor )
Argument Description
tabid The ID of the Control Center tab containing the object.
id The ID of the object.
fcolor The foreground color of the object.
Return value
Integer. Returns 0.
Usage
See ph.setccobj for more details.
