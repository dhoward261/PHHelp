ph_readfile PowerHome formula function
Description
Reads the contents of a file into memory and returns it as a string.
Syntax
ph_readfile ( filespec )
Argument Description
filespec String representing the path and filename of the file you wish to read.
Return value
String. Returns the contents of the file as a string.
Examples
The following examples demonstrate typical syntax/usage for this function.
• ph_readfile ("C:\powerhome\web\logtemp\last.csv")
• ph_readfile ("C:\powerhome\findMyPhone.txt")
